    GITEX Mobile Application Documentation

OVERVIEW

We are happy to present to you our first HTML Bootstrap Mobile application template. GITEX can be used to create your own application that could be similar to one of the best and very well known applications such as Uber, Lyft or Yandex.

Moreover, this template can also be used for other purposes such as car/bus tracking mobile applications or anything that might use the pickup and drop-off functionality incorporated in this wonderful template!

GITEX uses latest technologies such as HTML5, CSS3, JS - ES6. The implementation is fast & reliable. All the content can be easily modified without requiring special knowledge. Theme can be used for PhoneGap/Cordova applications as well. Note: If you might want to have the same design in PSD format - you can get one HERE

Since this is a ridesharing application there are separate parts involved for customization

  • Customer
  • Driver

We will go into details of how to edit the page and where you can find information for each page. You can find the information by pressing the DRIVER or PASSENGER dropdown in the main menu and choosing the appropriate page name.

If you have any questions regarding this template please do not hesitate to ask. Please leave your question in comments sections and our team will respond to it immediately. The best thing about our support it that we answer any question with video tutorial to make your lives easier and working processes smoother.
